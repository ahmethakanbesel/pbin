**Change 3 — Apply Basic Auth middleware to write endpoints only:**
Replace the two existing `mux.HandleFunc` registrations for POST /api/upload and POST /api/paste with versions wrapped in the middleware:

```go
mux.Handle("POST /api/upload", middleware.BasicAuth(cfg.Auth, http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {

        bucketHandler.Upload(w, r)
    } else {
        fileHandler.Upload(w, r)
    }
})))
mux.Handle("POST /api/paste", middleware.BasicAuth(cfg.Auth, http.HandlerFunc(pasteHandler.Create)))
```

Note: use `mux.Handle` (not `mux.HandleFunc`) because BasicAuth returns an `http.Handler`, not a function.
