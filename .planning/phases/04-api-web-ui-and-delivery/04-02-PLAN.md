Define:
```go
type UIHandler struct{}
func NewUIHandler() *UIHandler
func (h *UIHandler) Home(w http.ResponseWriter, r *http.Request)   // GET /
func (h *UIHandler) Paste(w http.ResponseWriter, r *http.Request)  // GET /paste
func (h *UIHandler) Bucket(w http.ResponseWriter, r *http.Request) // GET /bucket
```

UIHandler has no service dependencies — it only renders static HTML.

**Common page structure for all three handlers:**
- Call `securityHeaders(w)`.
- Set `Content-Type: text/html; charset=utf-8`.
- Set CSP: `default-src 'none'; script-src 'unsafe-inline' https://cdn.jsdelivr.net; style-src 'unsafe-inline' https://cdn.jsdelivr.net; form-action 'self'; connect-src 'self'`
  - Note: `connect-src 'self'` is required so the JS fetch calls to /api/* work.
- Render HTML via `fmt.Fprintf(w, ...)`.

**Home handler (file upload) — GET /:**
Page renders a file upload form. Uses JS fetch to POST multipart to `/api/upload`.
- Drag-and-drop drop zone: a `<div id="drop-zone">` that accepts dropped files and populates a hidden `<input type="file" id="file-input" name="file">`.
- Fields: file input, expiry select, optional password input, one-use checkbox.
- Submit button: "Upload".
- On success: show a result panel with URL (copy button), delete URL, expires_at.
- On error: show error message.

JS behavior (inline `<script>` tag):
```javascript
// Drop zone: dragover/dragleave/drop events on #drop-zone
// Drop sets #file-input.files and updates #drop-zone text to show filename
// Form submit: preventDefault, build FormData from the form, fetch('/api/upload', {method:'POST', body: formData})
// On 201: parse JSON, show #result panel with url, delete_url, expires_at
// On error: show #error-msg with response.error
```

**Paste handler — GET /paste:**
Page renders a paste creation form. Uses JS fetch to POST JSON to `/api/paste`.
- Fields: title (text, optional), content (textarea, required), language (select with common languages: plaintext, bash, c, cpp, css, go, html, java, javascript, json, markdown, python, ruby, rust, sql, typescript, yaml), expiry select, optional password input, one-use checkbox.
- Submit button: "Create Paste".
- On success: show result panel with URL, delete URL, expires_at.
- On error: show error message.

JS behavior:

// Form submit: preventDefault

// On error: show error
