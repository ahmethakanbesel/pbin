Run each check in a separate terminal:

1. **Health** (baseline):
```bash
curl -s http://localhost:8080/health
# Expected: {"status":"ok"}
```

2. **Basic upload + download**:
```bash
echo "hello world" > /tmp/test.txt
curl -s -F "file=@/tmp/test.txt" -F "expiry=1d" http://localhost:8080/api/upload
# Expected: {"url":"http://.../{slug}","delete_url":"http://.../delete/{slug}/{secret}","expires_at":"...","is_image":false}
# Save the slug from the URL. Then:
SLUG=<slug-from-above>
curl -v http://localhost:8080/$SLUG
# Expected: 200 with Content-Disposition: attachment; filename="test.txt" and body "hello world"
```

3. **Delete**:
```bash
SECRET=<secret-from-delete_url>
curl -s http://localhost:8080/delete/$SLUG/$SECRET
# Expected: {"deleted":true}
curl -s http://localhost:8080/$SLUG
# Expected: 404
```

4. **Expiry** (10-minute):
```bash
curl -s -F "file=@/tmp/test.txt" -F "expiry=10m" http://localhost:8080/api/upload
# Save slug. Check expires_at is ~10 minutes from now in RFC3339.
```

5. **One-use download**:
```bash
curl -s -F "file=@/tmp/test.txt" -F "expiry=1d" -F "one_use=1" http://localhost:8080/api/upload
SLUG=<slug>
curl -v http://localhost:8080/$SLUG
# Expected: 200 first download
curl -v http://localhost:8080/$SLUG
# Expected: 410 Gone
```

6. **Password protection**:
```bash
curl -s -F "file=@/tmp/test.txt" -F "expiry=1d" -F "password=secret123" http://localhost:8080/api/upload
SLUG=<slug>
curl -v http://localhost:8080/$SLUG
# Expected: 401 (no password provided) or HTML password form
curl -v -H "X-Password: wrong" http://localhost:8080/$SLUG
# Expected: 401
curl -v -H "X-Password: secret123" http://localhost:8080/$SLUG
# Expected: 200 with file content
```

7. **Image upload** (PNG or JPEG):
```bash
# Create a minimal valid PNG (1x1 pixel)
curl -s https://httpbin.org/image/png -o /tmp/test.png 2>/dev/null || \
  python3 -c "import sys; sys.stdout.buffer.write(bytes([137,80,78,71,13,10,26,10,0,0,0,13,73,72,68,82,0,0,0,1,0,0,0,1,8,2,0,0,0,144,119,83,222,0,0,0,12,73,68,65,84,8,215,99,248,207,192,0,0,0,2,0,1,227,70,220,190,0,0,0,0,73,69,78,68,174,66,96,130]))" > /tmp/test.png
curl -s -F "file=@/tmp/test.png" -F "expiry=1d" http://localhost:8080/api/upload
# Expected: is_image: true in response
SLUG=<slug>
curl -v http://localhost:8080/$SLUG
# Expected: Content-Disposition: inline (not attachment) + Content-Type: image/png
```

8. **Security headers** (on any response):
```bash
curl -v http://localhost:8080/health 2>&1 | grep -i "X-Content-Type-Options\|X-Frame-Options"
# Expected: both headers present
```

9. **Upload size limit**:
```bash
# Create a file slightly over 100 MB (the default limit)

PBIN_UPLOAD_MAX_BYTES=1024 go run ./cmd/pbin/ &

dd if=/dev/zero bs=2048 count=1 2>/dev/null | curl -s -F "file=@-;filename=big.bin" http://localhost:8080/api/upload
# Expected: 413 with error about file size
```
