**Create — handles POST /api/paste with JSON body:**
- Call `securityHeaders(w)`
- Decode JSON body into struct: `{content, title, lang, expiry, password, one_use bool}`
- Validate `content` not empty — return 400 if empty
- Default `expiry` to `"1d"` if empty
- Call `h.svc.Create(ctx, paste.CreateRequest{...})`
- Return 201 JSON:
```go
type pasteCreateResponse struct {
    URL       string  `json:"url"`
    DeleteURL string  `json:"delete_url"`
    ExpiresAt *string `json:"expires_at"` // RFC3339 or null
}
```

**View — GET /{slug} (paste) — HTML with highlight.js:**
- Call `securityHeaders(w)`
- Read password from `X-Password` header or `?password=` query param
- Call `h.svc.Get(ctx, slug, password)`
- On ErrWrongPassword: `servePasswordForm(w, slug)` (same function)
- On ErrExpired/ErrAlreadyConsumed: 410 JSON
- On ErrNotFound: 404 JSON
- On success: render HTML page using `fmt.Fprintf`

The HTML page must include:
1. CSP header: `default-src 'none'; script-src 'nonce-{nonce}' https://cdnjs.cloudflare.com; style-src 'nonce-{nonce}' https://cdnjs.cloudflare.com; img-src 'none'`
   - Generate a 16-byte crypto/rand nonce, base64-encode it, apply to all inline script/style tags and the CSP header
2. highlight.js CSS loaded from CDN (both light and dark themes with media query):
```html
<link rel="stylesheet" media="(prefers-color-scheme: light)"
      href="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.11.1/styles/github.min.css">
<link rel="stylesheet" media="(prefers-color-scheme: dark)"
      href="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.11.1/styles/atom-one-dark.min.css">
```
3. highlight.js script from CDN:
```html
<script src="https://cdnjs.cloudflare.com/ajax/libs/highlight.js/11.11.1/highlight.min.js"></script>
<script nonce="{nonce}">hljs.highlightAll();</script>
```
4. Metadata bar: title (if non-empty), language, created info, expiry info
5. Code block with the paste content HTML-escaped inside `<pre><code class="language-{lang}">`
   - Use `html.EscapeString(result.P.Content)` to prevent XSS — NEVER use `text/template` auto-escape here, escape manually and write with `fmt.Fprintf`
6. Line numbers via CSS counter (no JS): `pre { counter-reset: line; } pre code { counter-increment: line; }`
7. Copy button (one-click clipboard copy): `<button onclick="navigator.clipboard.writeText(document.querySelector('code').textContent)">Copy</button>`
8. Raw link: `<a href="/raw/{slug}">Raw</a>`

**Raw — GET /raw/{slug}:**
- Call `securityHeaders(w)`
- Read password from `X-Password` header or `?password=` query param
- Call `h.svc.Get(ctx, slug, password)`
- On error: same error mapping as View
- On success: set `Content-Type: text/plain; charset=utf-8`, write `result.P.Content` directly (do NOT HTML-escape for raw endpoint — it must return the original text)
- Set `Content-Disposition: inline; filename="{slug}.txt"`

**Delete — GET /delete/{slug}/{secret}:**
- Call `securityHeaders(w)`
- Extract `slug` and `secret` from `r.PathValue`
- Call `h.svc.Delete(ctx, slug, secret)`
- Return `{"deleted": true}` on 200
- ErrNotFound → 404, ErrBadDeleteSecret → 403

Note: `securityHeaders`, `writeJSON`, `writeError`, `servePasswordForm` are in the `handler` package already (from file.go). Do NOT redefine them.

Import `html` package for `html.EscapeString`. Import `crypto/rand` and `encoding/base64` for nonce generation.
